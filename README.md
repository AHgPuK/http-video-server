# http-video-server
An idea to watch a local video library on TV set which has only browser. Sony TV, for example.
In other words, video file sharing by http

#Prerequisites: 
Node.js, npm (https://www.nodejs.com)

#Install:
npm i http-video-server

#Run:
node index
